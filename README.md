# orskbot

Bot for Telegram channel #orsk

### Installation
Required python3

Crearte virtualenv
```sh
python -m venv .env
```

Activate virtualenv

Windows
```sh
C:\> .env\Scripts\activate
```

Linux and OS X 
```sh
$ source .env/Scripts/activate
```

Install dependens
```sh
pip install -r requirements.txt
```

### Run

Windows
```sh
bot_loop.cmd
```

Linux and OS X
```sh
$ source .env/bin/activate
$ python bot.py
```
